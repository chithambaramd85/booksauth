1. Git Commands- Build & Deploy
Steps:
1. Upload/Commit to Github: (in vscode terminal)
git init
git add .
git commit -m "my first commit"
git branch -M main
git remote add origin https://github.com/[user-name]/[repo-name].git
git push -u origin main

2. Build: (in vscode terminal)
npm install -g angular-cli-ghpages
ng build --prod --base-href "https://[user-name].github.io/[repo-name]/"
(or)
Angular 12 - new build
ng deploy --base-href=https://chithambaramd85.github.io/booksauth/ --name=chithambaramd85 --email=chithambaramd85@gmail.com
Deploy:
npx ngh --dir=dist/foldename-new
